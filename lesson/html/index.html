初めてのホームページ（HTML入門）

HTML とは

  • HTML はホームページを記述する言語です
  • HTML では主に文章の構造を記述します
  • WWW ブラウザが HTML を読み取って人がわかりやすいように表示します

ブラウザでいま表示しているページの HTML ソースコードを表示してみましょう
副ボタンをクリック（右クリック or ２本指でクリック）してメニューを表示し「ページのソースを表示」を選択します

主な機能

  • ページ内の文章と構造を記述する
  • 他のページヘのリンクを貼る
  • 画像を表示する

リンク

a タグでドキュメント間にリンクが貼れます

  • div タグ
  • HTML5 タグ

サブディレクトリ内のページへのリンク

  • サプページ1

URL とは

  • URL(Uniform Resource Locator)とは、インターネット上のリソース（資源）を特定するための書式です
  • 主にブラウザでホームページの在り処を指定する為に使用します

ホームページの作成

会社のホームページを作ってみましょう

  1. 以下のディレクトリ、ファイル構成とする
  2. index.html から、その他のページヘリンクを貼る
  3. インターネット上にある画像を表示する
  4. 文章構造が分かりやすいようにインデントを整える
※ 各ページの内容はシンプルで構いません（見出しとページ説明文等）
